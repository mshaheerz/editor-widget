Method 1: Auto-Init

Auto-Initialization with Data Attributes

The simplest way to use the widget. Just add data-rich-text-editor attribute to any element with an ID, and it will automatically initialize on page load.

<div id="editor1" data-rich-text-editor data-placeholder="Type here..." data-min-height="250px"></div>

Method 2: Programmatic

Programmatic Initialization

Use the initRichTextEditor() function for more control. Perfect when you need to initialize editors dynamically or with complex options.

const widget = initRichTextEditor('editor2', { placeholder: 'Custom placeholder...', minHeight: '250px', onChange: (content) => console.log(content) });

Method 3: Class-Based

Class-Based Initialization

Use the RichTextEditorWidget class for full control. This approach gives you access to lifecycle methods like mount(), unmount(), and update().
